# Security Hardening Checklist

1. Rotate all static credentials every 30 days.
2. Restrict all internal services to private network only.
3. Enforce TLS termination at edge gateway.
4. Enable Elasticsearch auth and role-based users.
5. Enable image scanning on each release candidate.
6. Configure secret manager integration for runtime secrets.
7. Enable audit logging retention for 90 days.
8. Configure WAF/rate limits at edge ingress.
